Welcome to GraphSketcher 2

GraphSketcher is a simple, elegant tool for quickly sketching graphs and plotting data — but you don’t even need data to get started. It’s perfect for reports, presentations, and problem sets where you need to produce sharp-looking graphs on the fly.

GraphSketcher 2.0 test (v45) — January 20, 2014

GraphSketcher 2 is the first open-source release and requires OS X 10.8. The following are changes from OmniGraphSketcher 1.2.4:

  • The app is now sandboxed.
  • Added support for OS X Auto Save.
  • Added support for OS X Versions and synced changes via OmniPresence.
  • Retired Support for Legacy Graphs: With the move to 64-bit, GraphSketcher is no longer able to read older .graph documents from Robin Stewart’s original GraphSketcher app.
  • Added support for OS X automatic graphics switching.
  • The application bundle ID is now com.graphsketcher.GraphSketcher.
  • Omni Group Licensing and Software Update frameworks have been eliminated.
  • Replaced the application icon with a version that doesn't include the Omni logo.
  • Replaced the Omni About panel with a generic one.
  • Miscellaneous code changes and improvements (archiving, memory management, etc).
  • Removed 99.9% of Omni Group & OmniGraphSketcher references; let us know if you find any that slipped by.
  • Open sourced GraphSketcher under our the same MIT-style license as our Objective-C Frameworks. All license references should reflect this license (again, please let us know).
